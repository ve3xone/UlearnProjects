Шаги решения:
Построение графа: Узлы графа — города, ребра — дороги между ними с весами, равными времени прохождения.
Распределение информации: Использование алгоритма Дейкстры для нахождения минимального времени доставки информации от столицы до всех остальных городов.

Пояснения:

Класс CityNetwork: Определяет сеть городов и дорог между ними.

Класс Edge: Представляет дорогу между двумя городами с расчетным временем прохождения.

Класс PriorityQueue<T>: Реализация простой очереди с приоритетом для алгоритма Дейкстры.

Метод AddRoad: Добавляет дорогу между двумя городами с учетом длины и скорости.

Метод CalculateMinimumTime: Реализует алгоритм Дейкстры для нахождения минимального времени распространения информации от столицы до всех городов.
